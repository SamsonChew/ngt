# Algorithm as Sovereignty

A short scroll-through companion piece to my NGT2001F presentation.

**The argument:** We call recommendation algorithms cosmopolitan and neutral, but we treat them as sovereign — as instruments of state power. The TikTok / Oracle deal of January 2026 made this visible.

## Viewing it

Open `index.html` in any modern browser. No build step, no install.

```
open index.html
```

Or serve it locally:

```
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Structure

- **Act I** — What an algorithm actually does (Labubu case)
- **Act II** — The asymmetry (interactive: Meta/TikTok, Samsung/Huawei)
- **Act III** — The January 2026 Oracle deal timeline
- **Closing** — The argument, in one sentence

## Stack

Plain HTML, CSS, and vanilla JavaScript. GSAP + ScrollTrigger are loaded from a CDN for scroll animations; everything degrades gracefully without them.
